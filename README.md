# Open-Think-Reflex

> **AI输入加速器** - 通过反射机制加速人与AI的交互

## 核心概念

> **AI = 大脑**（复杂思考）  
> **反射 = 快捷指令**（快速响应）

这个项目实现了一套反射形成系统，模仿人类形成反射的过程：
- **大脑** 处理复杂推理
- **反射** 处理高频操作
- 重复使用 → 形成反射
- 长期不用 → 自动衰减

## 应用场景

```
用户输入"用户API"
    │
    ▼
┌─────────────────────────────────────┐
│ 思维链树展开                        │
│ ├── 分页 pagination ───── 85%       │
│ ├── 响应 response ───── 70%        │
│ └── 错误 error ───────── 60%       │
└─────────────────────────────────────┘
    │
    ▼
用户选择"分页" → 按空格 → AI生成完整代码
```
